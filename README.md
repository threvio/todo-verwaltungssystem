# Todo-Verwaltungssystem

Ein einfaches JavaFX-Projekt zur Verwaltung von Aufgaben.

## Funktionen
- Aufgabe hinzufügen
- Aufgaben anzeigen
- Aufgabe löschen
- Speicherung in SQLite-Datenbank

## Architektur
- Model
- Repository (Schnittstelle)
- Service
- UI (JavaFX)

## Technologien
- Java 24
- JavaFX
- Maven
- SQLite
